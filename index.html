US NUMBER VALIDATOR

In the US, phone numbers can be formatted in many ways. Here are some examples of valid formats for US phone numbers:

  • +1 555-555-5555.
  • +1 (555) 555-5555.
  • 1(555)555-5555.
  • 1 555 555 5555.
  • 5555555555.
  • 555-555-5555.
  • (555)555-5555.

Where:

country code | area code | office code | subscriber number

To check if a number is a valid US number just enter the number here and press CHECK.

Please note that for US to US calls the country code can be ommited.
